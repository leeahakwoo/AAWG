# Usage Guide

## 1. Prerequisites

- Python 3.9+ 설치
- 가상 환경 사용 권장 (venv 또는 conda)
- 프로젝트 루트에서 가상 환경 생성 및 활성화:
  ```bash
  python -m venv .venv
  source .venv/bin/activate   # macOS/Linux
  .venv\Scripts\activate  # Windows
  ```
- 필수 패키지 설치:
  ```bash
  pip install -r requirements.txt
  ```

## 2. Installation Installation
